# prompt-automat-framework
Interactive prompt builder for the AUTOMAT framework — a 7-step structured approach to writing effective AI prompts (Act as, User persona, Targeted action, Output, Mode, Atypical cases, Topic whitelisting). Available as a standalone HTML page and a React component.

## How to use the application
Donwload both files in the same directory and start the aplication from the HTML file.
Should be compatible with all browser and responsive (even though i recommend to use it on a larger screen)
